Crimeno

Crimeno was developed to help tackle the increasing recurring crimes in particular areas. A RESTful MERN stack app, centred around the Police and Google Maps API. Users can pinpoint a location where a crime was committed using google maps markers. Other users will then be made aware of these crimes if they search a route within that area of the crime. There is also a heatmap derived from the Police API that highlights the crime in areas.
